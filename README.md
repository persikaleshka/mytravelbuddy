# mytravelbuddy

## Настройка проекта

### Установка зависимостей
```bash
cd frontend
npm install
```

### Настройка переменных окружения
Создайте файл `.env` в директории `frontend` со следующим содержимым:
```
VITE_YANDEX_MAPS_API_KEY=ваш_ключ_API_Яндекс_Карт
```

Для получения API ключа Яндекс Карт:
1. Перейдите на сайт https://developer.tech.yandex.ru/
2. Создайте аккаунт или войдите в существующий
3. Создайте новое приложение
4. Выберите сервис "JavaScript API и HTTP Геокодер"
5. Скопируйте API ключ и вставьте в файл `.env`

### Запуск проекта
```bash
cd frontend
npm run dev
```
